# Vue的复用弊端

# 当俩个相同的标签来回切换的时候内部的value值不会改变

# 例如：

<span v-if="aa"> 		/* aa=true */
<input type="text">
</span>
<span v-else>
<input type="text">
</span>
<button @click="aa=!aa">
  切换
</button>

# 当改变其中一个input的value切换标签时会保留value，需加入标签属性key来区分俩个为不同的input.value

# 例如

<span v-if="aa"> 		/* aa=true */
<input type="text" key="inp1">
</span>
<span v-else>
<input type="text" key="inp2">
</span>
<button @click="aa=!aa">
  切换
</button>
